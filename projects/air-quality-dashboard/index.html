Air Quality Dashboard

This Python-based project uses the AQICN (World Air Quality Index) API to fetch real-time air quality data for major cities across the USA and India. All data is stored in MongoDB Atlas and visualized using a Streamlit web dashboard, featuring interactive maps, graphs, and forecasts for AQI, pollutants, and weather metrics. The project can be automated via GitHub Actions to continuously update readings and maintain live dashboards.

About

Short description: A Python-powered dashboard that taps the AQICN (World Air Quality Index)￼ API to monitor live air quality (AQI, pollutants, weather metrics) across major cities in the USA and India. Data is ingested into MongoDB Atlas for persistent storage, and seamlessly visualized via a Streamlit web app — including interactive maps, trend charts.
